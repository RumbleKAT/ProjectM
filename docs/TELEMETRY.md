# Telemetry

The platform includes a telemetry system to monitor application performance, usage metrics, and errors.

## Features
- **Metrics Collection**: Tracks key events and system performance indicators.
- **Logging**: Centralized logging for debugging and auditing.
- **Monitoring**: Provides data for observing the health of the RAG system and the Telegram bot.

The telemetry implementation is located in `server/utils/telemetry/`.
